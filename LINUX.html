Linux daily use commands

-- File and Directory Management ðŸ‘‡

ls: List directory contents.

ls -l: Detailed listing.

ls -a: List all files, including hidden ones.

cd: Change directory.

cd .. : Move to the parent directory.

cd ~ : Move to the home directory.

pwd: Print working directory.

mkdir: Make directory.

mkdir folder_name: Create a new folder.

rm: Remove files or directories.

rm filename: Remove a file.

rm -r directory_name: Remove a directory and its contents recursively.

cp: Copy files or directories.

cp source_file destination_file: Copy a file.

mv: Move or rename files or directories.

mv old_name new_name: Rename a file or directory.

mv source destination: Move a file or directory.

-- File Viewing and Editing ðŸ‘‡

cat: Concatenate and display file content.

less: View file content page by page.

head: Display the beginning of a file.

tail: Display the end of a file.

nano: Simple text editor.

nano filename: Open a file for editing.

vim or vi: Powerful text editor.

vim filename: Open a file for editing.

-- System Information and Monitoring ðŸ‘‡

top: Display Linux processes.

ps: Display currently running processes.

ps aux: Detailed process listing.

df: Show disk space usage.

df -h: Human-readable format.

free: Display amount of free and used memory in the system.

-- Package Management ðŸ‘‡

apt: Advanced Package Tool for managing packages.

sudo apt update: Update package lists.

sudo apt upgrade: Upgrade installed packages.

sudo apt install package_name: Install a new package.

sudo apt remove package_name: Remove a package.

sudo apt autoremove: Remove packages that are no longer needed.

-- Network Management ðŸ‘‡

ifconfig: Display or configure network interface parameters.

ping: Send ICMP ECHO_REQUEST to network hosts.

ping hostname: Ping a specific host.

ssh: Secure Shell for securely accessing remote systems.

ssh username@hostname: Connect to a remote host.

-- User and Group Management ðŸ‘‡

useradd: Create a new user.

userdel: Delete a user.

passwd: Change user password

groups: Display groups a user is in.

adduser: Interactive utility for adding a new user.

deluser: Interactive utility for deleting a user
